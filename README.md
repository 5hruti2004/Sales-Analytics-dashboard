# Sales-Analytics-dashboard
The dashboard provides insights into revenue trends, product performance, regional sales distribution, and salesperson effectiveness.

## Project Overview

This project analyzes sales performance using SQL and Power BI. The dashboard provides insights into revenue trends, product performance, regional sales distribution, and salesperson effectiveness.

## Tools Used

* MySQL
* Power BI
* Microsoft Excel
* GitHub

## Dashboard Features

* Total Revenue Analysis
* Region-wise Sales Performance
* Product Performance Analysis
* Top Salespersons
* Manager Performance Analysis
* Interactive Filters and Slicers

## Dashboard Pages

### Page 1: Sales Analytics Dashboard

* KPI Cards
* Revenue Trend Analysis
* Region-wise Revenue Distribution
* Manager Performance

### Page 2: Product & Sales Performance

* Top Products
* Top Salespersons
* Product Contribution
* Business Insights
